Redis 使用标准的 major.minor.patchlevel版本做法。 stable表示稳定版本，一般用偶数版本，如：1.2, 2.0, 2.2, 2.4, 2.6。 奇数标识不稳定版本，如：2.7.x，他未来的稳定版本是2.8。

2.6.16 稳定版 这是最新的Redis 2.4版本的替换版本，最新的Redis 2.6版本支持Lua脚本，时间能精确到毫秒，改进了内存使用情况，无限制的客户端连接数量， AOF改进，更好的性能和一些新的命令功能。完整的新功能列表和修订内容请参考 发布日志

安装

下载，解压和安装：

$ wget http://redis.googlecode.com/files/redis-2.6.16.tar.gz
$ tar xzf redis-2.6.16.tar.gz
$ cd redis-2.6.16
$ make

编译后的可执行文件在src目录中，可以使用下面的命令运行Redis:

$ src/redis-server

你可以使用内置的客户端连接Redis:

$ src/redis-cli
redis> set foo bar
OK
redis> get foo
"bar"

Redis集群在哪里？

Redis Cluster, the distributed version of Redis, is making a lot of progresses and will be released as beta at the start of Q3 2013, and in a stable release before the end of 2013. You can watch a video about what Redis Cluster can currently do. The source code of Redis Cluster is publicly available in the unstable branch, check the cluster.c source code.
