时间复杂度： O（N），N为一些脚本检查（检查一个简单的脚本是一个O（1）操作）。

返回脚本在脚本缓存中的存在信息。

这个命令可以是一个或者多个SHA1串，返回一个0或者1的列表，标识脚本是否在脚本缓存中。 This can be useful before a pipelining operation to ensure that scripts are loaded (and if not, to load them using SCRIPT LOAD) so that the pipelining operation can be performed solely using EVALSHA instead of EVAL to save bandwidth.

关于使用 Redis 对 Lua 脚本进行求值的更多信息，请参见 EVAL 命令。

Return value

Multi-bulk reply 包含 0 和 1 ，前者表示脚本不存在于缓存，后者表示脚本已经在缓存里面了。 列表中的元素和给定的 SHA1 校验和保持对应关系，比如列表的第三个元素的值就表示第三个 SHA1 校验和所指定的脚本在缓存中的状态。
